# Resolución de 29 de diciembre de 2022, de la Comarca de Cinco Villas (Zaragoza), referente a la convocatoria para proveer varias plazas.

En el «Boletín Oficial de la Provincia de Zaragoza», número 296, de 28 de diciembre de 2022, se han publicado íntegramente las bases que han de regir la convocatoria de las siguientes plazas incluidas en la OEP para la Estabilización del Empleo Temporal:

Una plaza de Coordinador de Servicios Sociales, del grupo A2, nivel 26, mediante el sistema de concurso.

Una plaza de Educador Social, del grupo A2, nivel 2, mediante el sistema de concurso.

Cinco plazas de Trabajador Social, del grupo A2, nivel 21, mediante el sistema de concurso.

Una plaza de Coordinador de Turismo, Cultura y Deportes, del grupo A2, nivel 21, mediante el sistema de concurso.

Una plaza de Técnico de RSU, del grupo A2, nivel 21, mediante el sistema de concurso.

Una plaza de Técnico de Deportes, del grupo C1, nivel 19, mediante el sistema de concurso.

Una plaza de Técnico Auxiliar Deportivo, del grupo C2, nivel 17, mediante el sistema de concurso.

Una plaza de Conductor de Transporte Social, del grupo C2, nivel 17, mediante el sistema de concurso.

Tres plazas de Auxiliar Administrativo, del grupo C2, nivel 17, mediante el sistema de concurso.

Tres plazas de Auxiliar de Ayuda a Domicilio, de la Zona de Sos, del grupo C2, nivel 17, mediante el sistema de concurso.

Diez plazas de Auxiliar de Ayuda a Domicilio, de la Zona de Luna, del grupo C2, nivel 17, mediante el sistema de concurso.

Tres plazas de Auxiliar de Ayuda a Domicilio, de la Zona de Tauste, del grupo C2, nivel 17, mediante el sistema de concurso.

Siete plazas de Auxiliar de Ayuda a Domicilio, de la Zona de Sádaba, del grupo C2, nivel 17, mediante el sistema de concurso.

Una plaza de Trabajador Social, del grupo A2, nivel 21, mediante el sistema de concurso-oposición.

Una plaza de Administrativo, del grupo C1, nivel 19, mediante el sistema de concurso-oposición.

Dos plazas de Conductor de Transporte Social, del grupo C2, nivel 17, mediante el sistema de concurso-oposición.

Una plaza de Auxiliar Administrativo, del grupo C2, nivel 17, mediante el sistema de concurso-oposición.

Seis plazas de Auxiliar de Ayuda a Domicilio, de la Zona de Tauste, del grupo C2, nivel 17, mediante el sistema de concurso-oposición.

Dos plazas de Auxiliar de Ayuda a Domicilio, de la Zona de Sádaba, del grupo C2, nivel 17, mediante el sistema de concurso-oposición.

Se abre el plazo de presentación de solicitudes, que será de veinte días hábiles a contar desde el día siguiente a la publicación de este anuncio en el «Boletín Oficial del Estado».

Ejea de los Caballeros, 29 de diciembre de 2022.–El Presidente, Santos Navarro Giménez.
