# Resolución de 27 de diciembre de 2022, del Ayuntamiento de Pozo Alcón (Jaén), referente a la convocatoria para proveer varias plazas.

En el «Boletín Oficial de la Provincia de Jaén», número 246, de 23 de diciembre de 2022, se ha publicado anuncio sobre las bases que han de regir la convocatoria para proveer las siguientes plazas de personal laboral fijo.

Una plaza de Delineante, personal laboral fijo, subgrupo C1, mediante el sistema de concurso.

Una plaza de Auxiliar Administrativo, personal laboral fijo, subgrupo C2, mediante el sistema de concurso.

Dos plazas de Conserje, personal laboral fijo, grupo E (Agrup. Prof.), mediante el sistema de concurso.

Una plaza de Auxiliar de Biblioteca, personal laboral a tiempo parcial, subgrupo C2, mediante el sistema de concurso.

Una plaza de Limpiadora, personal laboral fijo a tiempo parcial, grupo E (Agrup. Prof.), mediante el sistema de concurso.

Dos plazas de Guarda de Coto de Pesca, personal laboral fijo, grupo E (Agrup. Prof.), mediante el sistema de concurso.

Una plaza de Directora de Centro de Discapacitados, personal laboral fijo, subgrupo A2, mediante el sistema de concurso.

Una plaza de Monitor Centro de Discapacitados, personal laboral fijo, subgrupo A2, mediante el sistema de concurso.

Una plaza de Monitor de Centro Guadalinfo, personal laboral fijo, subgrupo C1, mediante el sistema de concurso.

Tres plazas de Oficial de Obras, personal laboral fijo, subgrupo C2, mediante el sistema de concurso.

Una plaza de Peón de Obras, personal laboral fijo, grupo E (Agrup. Prof.), mediante el sistema de concurso.

Una plaza de Fontanero/Electricista, personal laboral fijo, subgrupo C2, mediante el sistema de concurso.

Una plaza de Fontanero, personal laboral fijo, subgrupo C2, mediante el sistema de concurso-oposición.

Una plaza de Oficial de 2.ª de Instalaciones, personal laboral fijo, subgrupo C2, mediante el sistema de concurso-oposición.

Cinco plazas de Socorrista Piscina, personal laboral fijo discontinuo, grupo E (Agrup. Prof.), mediante el sistema de concurso-oposición.

Tres plazas de Portero de Piscina, personal laboral fijo discontinuo, grupo E (Agrup. Prof.), mediante el sistema de concurso-oposición.

Una plazas de Peón de Mantenimiento de Piscina, personal laboral fijo discontinuo, grupo E (Agrup. Prof.), mediante el sistema de concurso-oposición.

Una plazas de Guardarropa/Limpieza de Piscina, personal laboral fijo discontinuo, grupo E (Agrup. Prof.), mediante el sistema de concurso-oposición.

Una plaza de Lector/a de Contadores, personal laboral fijo discontinuo, grupo E (Agrup. Prof.), mediante el sistema de concurso.

Una plaza de Cocinera/Limpiadora de Escuela Infantil, personal laboral fijo, grupo E (Agrup. Prof.), mediante el sistema de concurso-oposición.

Una plaza de Limpiadora de Edificios, personal laboral fijo, grupo E (Agrup. Prof.), mediante el sistema de concurso-oposición.

Una plaza de Limpiadora de Escuela Infantil, personal laboral fijo a tiempo parcial, grupo E (Agrup. Prof.), mediante el sistema de concurso-oposición.

El texto íntegro de las bases, estarán expuestas en el tablón de anuncios y en la sede electrónica del Ayuntamiento de Pozo Alcón, en la dirección https://www.pozoalcon.es/portal-de-transparencia/empleados-publicos/procesos-de-seleccion-abiertos.html.

El plazo de presentación de solicitudes será de veinte días hábiles a contar desde el día siguiente a la publicación del anuncio en el «Boletín Oficial del Estado».

Los sucesivos anuncios referentes a esta convocatoria, cuando procedan, de conformidad con las bases, se harán públicos en el tablón de anuncios y la sede electrónica del Ayuntamiento de Pozo Alcón.

Pozo Alcón, 27 de diciembre de 2022.–El Alcalde, Iván Raúl Cruz Lozano.
