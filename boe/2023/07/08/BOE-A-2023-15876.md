# Resolución de 28 de junio de 2023, del Ayuntamiento de Silla (Valencia), referente a la convocatoria para proveer varias plazas.

En el «Boletín Oficial de la Provincia de València» número 218, de 14 de noviembre de 2022, se han publicado íntegramente las bases que han de regir la convocatoria para proveer varias plazas mediante concurso y concurso-oposición, en turno de estabilización:

Una plaza de Técnico/a de promoción, naturaleza funcionarial, Administración Especial, clase Cometidos Especiales, Titulados Superiores, mediante el sistema de concurso.

Dos plazas de Administrativo/a, naturaleza funcionarial, Administración General, subescala Administrativa, mediante el sistema de concurso.

Una plaza de Auxiliar de Biblioteca, naturaleza funcionarial, escala Administración Especial, clase Cometidos Especiales, Auxiliares Técnicos/as, mediante el sistema de concurso.

Una plaza de Psicopedagogo/a, naturaleza funcionarial, Administración Especial, subescala Técnica Superior, mediante el sistema de concurso.

Una plaza de Agente de Desarrollo Local, naturaleza funcionarial, Administración Especial, clase Cometidos Especiales, Diplomatura/Técnico/a Medio, mediante el sistema de concurso.

Una plaza de Oficial de Jardinería, naturaleza funcionarial, Administración Especial, clase Personal de Oficios, Oficial/a, mediante el sistema de concurso.

Una plaza de Técnico/a de Administración General, naturaleza funcionarial, Administración General, subescala Técnica, mediante el sistema de concurso.

Una plaza de Ayudante/a de Servicios Sociales, naturaleza funcionarial, Administración Especial, subescala Servicios Especiales, clase Cometidos Especiales, ayudante, mediante el sistema de concurso.

Una plaza de Ayudante/a de Cementerio, naturaleza funcionarial, Administración Especial, subescala Servicios Especiales, clase Personal Oficios, Oficial/a, mediante el sistema de concurso.

Seis plazas de Auxiliar Administrativo/a, naturaleza funcionarial, Administración General, subescala Auxiliar, mediante el sistema de concurso.

Tres plazas de Oficial/a de Polideportivo, naturaleza funcionarial, Administración Especial, subescala Servicios Especiales, clase Personal de Oficios, Oficial/a, mediante el sistema de concurso.

Una plaza de Monitor/a de Ejercicio Físico Regular, naturaleza laboral, de la plantilla de personal fijo discontinuo, tiempo parcial, grupo profesional Personal Técnico/a Auxiliar, mediante el sistema de concurso.

Una plaza de Operario/a de Servicios, naturaleza laboral, de la plantilla de personal fijo, grupo profesional de Operarios/as, mediante el sistema de concurso.

Una plaza de Operario/a de Servicios Múltiples, naturaleza laboral, de la plantilla de personal fijo, grupo profesional de Oficios, mediante el sistema de concurso.

Una plaza de Operario/a de Servicios Múltiples, naturaleza laboral, de la plantilla de personal fijo discontinuo, grupo profesional de Operarios/as, mediante el sistema de concurso.

Una plaza de Monitor/a de Manualidades, naturaleza laboral, de la plantilla de personal fijo discontinuo, tiempo parcial, grupo profesional Personal Técnico/a Auxiliar, mediante el sistema de concurso.

Una plaza de Trabajador/a Social, naturaleza funcionarial, Administración Especial, subescala Técnica, Media, mediante el sistema de concurso.

Una plaza de Psicólogo/a, naturaleza funcionarial, Administración Especial, subescala Técnica, Superior, mediante el sistema de concurso.

Una plaza de Monitor/a, naturaleza funcionarial, Administración Especial, clase Cometidos Especiales, Auxiliares Técnicos/cas, mediante el sistema de concurso.

Dos plazas de Monitor/a de Centro de Día, naturaleza laboral, Administración Especial, de la plantilla de personal fijo, tiempo parcial, grupo profesional Técnicos/as Auxiliares, mediante el sistema de concurso.

Una plaza de Operario/a de Servicios y Coto, naturaleza laboral, Administración Especial, de la plantilla de personal fijo, grupo profesional de Operarios/as, mediante el sistema de concurso.

Tres plazas de Operarios/as de Servicios y Coto, naturaleza laboral, Administración Especial, de la plantilla de personal fijo discontinuo, grupo profesional de Operarios/as, mediante el sistema de concurso.

Dos plazas de Vigilante/a Rural, naturaleza funcionarial, Administración Especial, subescala Servicios Especiales, clase Cometidos Especiales, Oficial/a, mediante el sistema de concurso.

Una plaza de Subalterno/a, naturaleza funcionarial, Administración General, subescala Subalterna, mediante el sistema de concurso-oposición.

Una plaza de Arquitecto/a Técnico/a, naturaleza funcionarial, Administración Especial, subescala Técnica, Media, mediante el sistema de concurso-oposición.

Una plaza de Ayudante/a de Servicios Sociales, naturaleza funcionarial, Administración Especial, subescala Servicios Especiales, clase Cometidos Especiales, Ayudante, a tiempo parcial, mediante el sistema de concurso-oposición.

Una plaza de Ingeniero/a Técnico/a industrial, naturaleza funcionarial, Administración Especial, subescala Técnica, Media, mediante el sistema de concurso-oposición.

Una plaza de Oficial de Vías y Obras, naturaleza funcionarial, Administración Especial, subescala Servicios Especiales, clase Personal de Oficios, Oficial/a, mediante el sistema de concurso-oposición.

Una plaza de Profesor/a de Escuela Permanente de Adultos, naturaleza laboral, de la plantilla de personal fijo discontinuo, grupo profesional Personal Técnico/a Medio, mediante el sistema de concurso-oposición.

El plazo de presentación de solicitudes será de diez días hábiles a contar desde el siguiente al de la publicación de esta resolución en el «Boletín Oficial del Estado».

Los sucesivos anuncios referentes a esta convocatoria, cuando procedan de conformidad con las bases, se harán públicos en la forma establecida en las citadas bases.

Silla, 28 de junio de 2023.–El Alcalde, Vicente Zaragozá Alberola.
